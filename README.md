## Getting Started :

- Install [node.js]()
- Install in command lines :
  - `npm install webpack -g`.
  - `npm install webpack-dev-server -g`.
- Start server `npm start`.
- Go to [http://localhost:8080/webpack-dev-server/bundle](http://localhost:8080/webpack-dev-server/bundle)
